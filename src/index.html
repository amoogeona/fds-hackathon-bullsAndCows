Bulls and Cows Game

Welcome to Amoogeona's

Play this easy, simple, peasy, lemon squeezy game!

1. Enter 3 digit numbers by avoiding duplication.
2. Guess the randomly chosen 3 digit numbers by AI.
3. Bulls = correct number, but wrong position.
4. Cows = correct number and correct position.
5. Make 3 Cows, you win this game.
6. Be careful, you must guess the numbers
within the certain time range.
7. Also, you have only 9 maximum trials on each game.

Good luck!
